# AndroidProject

Фадеев Егор ИКС-432

- В проекте реализована модель случайного блуждания (Random Walk) в двумерном пространстве с координатами (x, y) для группы из 14 человек. 
- Каждый участник за каждый шаг симуляции случайно меняет свои координаты.


## 1. Основы ООП. "Ходячий"
- Реализован класс `Human` с движением

## 2. Наследование
- Добавлен дочерний класс `Driver` который наследует методы и поля класса от родительского класса `Human`

## 3. Интерфейсы
- Реализован интерфейс `Movable` 
- В симуляции все объекты вызывают метод `move()` через ссылку типа `Movable`
