# AndroidProject

Фадеев Егор ИКС-432

**`main.kt`**
Создает список объектов `Human` и один объект `Driver`. 
Объединяет их в одну коллекцию и вызывает метод `move()` для каждого объекта. 
Добавляет задержку в 6 секунд, чтобы дать время для выполнения симуляции в потоках. 

**`classes/Human.kt`**
Содержит класс `Human`, представляющий человека. 
Наследуется от интерфейса `Movable`. 
Реализует метод `move()`, который моделирует случайное (Random Walk) движение в отдельном потоке (`Thread`). 
Использует `@Synchronized fun log` для вывода информации. 

**`classes/Driver.kt`**
Содержит класс `Driver`, представляющий водителя. 
Наследуется от класса `Human` и реализует интерфейс `Movable`. 
Переопределяет метод `move()`, который моделирует прямолинейное движение (Gauss-Markov Mobility Model) в отдельном потоке (`Thread`). 
Использует `@Synchronized fun log` для вывода информации. 

**`interfaces/Movable.kt`**
Содержит интерфейс `Movable`. 
Определяет общие свойства (`speed`, `x`, `y`) и метод (`move()`), необходимые для движения.
